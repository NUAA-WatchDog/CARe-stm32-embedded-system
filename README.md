## About

Product of 7th [*China Sofeware Cup*](http://www.cnsoftbei.com/) competition. The embedded system part of *CARe*.

The embedded system is used for connecting the sensors, collecting user data, and transferring to Android client of *CARe* through *bluetooth* module. It is developed on [*Keil Î¼Vision*](http://www2.keil.com/mdk5/uvision) IDE with C and [*CMSIS*](https://www.arm.com/why-arm/technologies/cmsis) Lib. The program runs on *STM32-F10x*.
